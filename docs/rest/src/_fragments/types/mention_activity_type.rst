Mention Activity
----------------

A mention activity indicates that the user has mentioned another user, perhaps via an @user syntax.  Mention has the following key/value pairs in its Data object:

UserId
    A **string** containing the user ID of the the person being mentioned, equivalent to the Id property of the :ref:`person object <person type>`.

UserProvider
    A **string** containing the user provider of the the person being mentioned, equivalent to the Provider property of the :ref:`person object <person type>`.
